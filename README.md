
# 🥋 유도 일지 (Judo Diary)

https://ldhishere.github.io/judo-diary/
유도 수련 기록을 체계적으로 관리하고 시각적으로 확인할 수 있는 개인 프로젝트입니다.

## ✨ 주요 기능

- **📅 캘린더**: 날짜별로 수련 여부를 한눈에 확인하고 일지를 작성할 수 있습니다.
- **🥋 기술 라이브러리**: 메치기와 굳히기 등 다양한 유도 기술을 검색하고 즐겨찾기에 추가할 수 있습니다.
- **📊 통계 분석**: 월별 수련 일수, 기술별 사용 빈도, 컨디션 및 운동 강도 추세를 차트로 보여줍니다.
- **💾 데이터 관리**: JSON 형식을 전용 백업 파일로 내보내거나 불러올 수 있어 기기 변경 시에도 데이터를 안전하게 지킵니다.
- **📱 PWA 지원**: 모바일 웹 앱으로 설치하여 언제 어디서든 간편하게 사용할 수 있습니다.

## 🛠 사용된 기술 (Tech Stack)

- **Frontend**: React (v19)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, motion
- **Icons**: Lucide React
- **Date Handling**: date-fns
- **Platform**: Web (Responsive Design)
